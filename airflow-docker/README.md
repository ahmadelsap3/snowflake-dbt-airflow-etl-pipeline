# Apache Airflow Docker Setup

This project provides a Docker-based setup for Apache Airflow, allowing you to easily run and manage workflows in a containerized environment.

## Project Structure

- **docker-compose.yaml**: Defines the services, networks, and volumes for the Apache Airflow setup.
- **Dockerfile**: Contains instructions to build the Docker image for Apache Airflow.
- **requirements.txt**: Lists the Python packages required for the Airflow project.
- **dags/**: Contains Directed Acyclic Graphs (DAGs) for defining workflows.
  - **example_dag.py**: An example DAG showcasing how to create workflows in Airflow.
- **logs/**: Directory for storing logs generated by Airflow tasks and services.
- **plugins/**: Directory for custom plugins that extend the functionality of Apache Airflow.
  
## Getting Started

### Prerequisites

- Docker
- Docker Compose

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/microsoft/vscode-remote-try-dab.git
   cd airflow-docker
   ```

2. Build and start the services:
   ```
   docker-compose up --build
   ```

3. Access the Airflow web interface at `http://localhost:8080`.

### Usage

- Place your DAG files in the `dags/` directory.
- Monitor logs in the `logs/` directory.
- Add custom plugins in the `plugins/` directory.
